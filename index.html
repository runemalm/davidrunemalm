OpenDDD.NET

An open-source toolchain/framework for .NET, currently in its alpha stage. This project aims to support the development of applications by providing core DDD building blocks, tools and strategies.

  • ✅ Implements key DDD principles such as aggregates, entities, and value objects.
  • ✅ Can be used in it's current stage as toolchain to pick and choose parts, such as repositories and event publishing components.
  • ✅ Focused on establishing a streamlined framework that enables building software using best-practice DDD principles.
GitHub Documentation
