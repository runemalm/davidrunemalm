Open Banking Proof-of-Concept Client

This proof-of-concept client simplifies open banking integration. The backend, built in C#/.NET, is designed to streamline onboarding for new developers working on bank integrations. It includes a JavaScript client SDK for seamless integration into websites. With a well-defined session concept and a structured state machine, the design ensures clarity and accelerates the development of bank integrations across various regions. The POC follows a monolithic architecture with clearly defined logical boundaries, enabling an effortless transition to a microservices architecture when needed.

  • ✅ Monolithic architecture for simplicity.
  • ✅ Scalability-ready design.
  • ✅ Focused approach: one developer per bank integration.
  • ✅ JavaScript SDK for quick and easy website integration.
