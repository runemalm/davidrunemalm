Open Banking POC

This project demonstrates how to design software to support scalable team operations. The backend is structured to manage Open Banking sessions, while the drop-in JavaScript client allows for seamless integration into any website.

A key advantage of this design is its scalability: adding a developer to work on a specific bank integration is simple and efficient. This approach is ideal for teams having to work on multiple bank integrations across different countries.

  • ✅ Backend designed for team scalability and modularity.
  • ✅ Supports new bank integrations with minimal effort.
  • ✅ Drop-in JavaScript client for fast and easy website integration.
